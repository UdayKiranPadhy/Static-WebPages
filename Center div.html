Center Stuff With CSS
5 Quick Ways

Method 1

RESULT:

HTML:

<div class="container">
    <div class="box"></div>
</div>

CSS:

.container{
    position: relative;
    height: 200px;
    width: 800px;
}    
.box{
    height: 80px;
    width: 80px;
    background-color:#20C4D5;
    position: absolute;
    margin: auto;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
}
